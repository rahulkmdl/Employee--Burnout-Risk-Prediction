# Employee--Burnout-Risk-Prediction
Machine Learning project to predict employee burnout risk using Linear Regression and Streamlit.# Employee Burnout Risk Prediction System

## Project Overview

This project uses Machine Learning to predict employee burnout risk based on workplace and lifestyle factors.

## Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Scaling using StandardScaler
* Linear Regression Model
* Model Evaluation (MAE, RMSE, R²)
* Streamlit Web Application

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* Joblib

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Train-Test Split
4. Feature Scaling
5. Model Training
6. Model Evaluation
7. Deployment with Streamlit
